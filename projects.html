Portfolio projects

Computational chemistry workflows, analysis scripts, and research software.

These projects demonstrate practical scientific software development, molecular simulation analysis, quantum-chemical post-processing, and workflow design for computational chemistry.

chemistry-analysis-tools

Python · MDAnalysis · Open Babel · portfolio-ready analysis scaffold

Lightweight computational chemistry utility repository containing MOL2 cleanup/charge comparison tools and molecular-simulation analysis templates. Current scaffold scripts include radial density analysis, radius-shell counts, and interfacial orientation metrics relative to a reference center of mass.

trajectory analysis radial density hydration counts orientation metrics

polymer-tool

Python · RDKit · Packmol · polymer/nanoparticle preparation

A PLGA-first model-preparation workflow for chemically traceable polymer chain construction, nanoparticle seed packing, packing assessment, provenance, and downstream handoff bundles. The project demonstrates how chemistry-aware model-building workflows can be organized into a command-line and GUI-accessible tool.

PLGA nanoparticle setup Packmol workflow design

qctddft

Python · Q-Chem · TDDFT · spectral-region analysis

A command-line toolkit for post-processing Q-Chem TDDFT calculations. The workflow extracts excitation data, builds broadened spectra, identifies spectral regions, assigns excited states, and clusters representative molecular structures for interpretation.

TDDFT Q-Chem spectral analysis clustering

AmberMD-Scripting

Shell scripting · AMBER · molecular dynamics setup

Shell-based utilities for AMBER molecular dynamics preparation and execution workflows. This repository supports the broader portfolio theme of reproducible, scriptable simulation setup.

AMBER Bash simulation setup
